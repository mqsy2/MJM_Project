4. System Logic (The Flow) 
(Explain step-by-step what happens when the user interacts with the system.) 
1. Input: User types "The sun is glaring on my screen" into the dashboard OR Sensor reads 
"Light Level: 950 (Direct Sunlight)". 
2. Processing: PHP combines this text and sensor data into a prompt and sends it to the 
Google Gemini API. 
3. AI Decision: The AI replies with a JSON command to CLOSE the curtains at full speed 
(255) to reduce the detected glare and heat. 
4. Storage: PHP saves this decision (Status: CLOSE, Speed: 255) to the MySQL table 
device_logs (or command_queue). 
5. Actuation: Arduino reads the database via WiFi and turns on the 12V DC Motor via the 
XY-160D driver to physically move the curtain.
